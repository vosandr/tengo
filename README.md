Всем здравствуйте. Данное приложение создано для обработки информации и создания информационной базы знаний.  

На данный момент приложение будет иметь 2 режима:
Режим просмотра. Можно отображать элементы в папке и содержимое папки. Сейчас работает более-менее стабильно. Я думаю, что даже с помощью этого решения можно просматривать базы информации, создавая их в других приложениях, наподобие vscode, obsidian или обычный текстовый редактор и файловый менеджер.  

Режим редактирования. Будет стоить 600 рублей/месяц. Пока недоступен для включения. Включить можно будет в будущем в настройках. Позволит создавать, удалять элементы в папке, переименовывать файлы и редактировать содержимое файла. В принципе, стандартный набор в других системах редактирования файлов и папок, однако, так как специализация этого приложения будет идти в направлении удобства использования при создании и редактировании информационной базы знаний, в будущем эта цена может показаться более оправданной
